# SHRT

URL-Shortner and LinkTree alternative using ReactJS, NextJS, GCP (Firebase), Emotion, and Framer Motion

# Features

- URL shortening and analytics
- Personal Landing Page

## Installation

Be sure to also the [Acto](https://gitlab.com/gvempire/acto) repository to control the firebase app `gv-collection`.

Use the package manager [yarn](https://classic.yarnpkg.com/en/) to install the packages and dependencies.

```bash
yarn
```
